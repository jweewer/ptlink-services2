 TS usage guide 
----------------
user 
  first_TS  -> from nick introduction
  local_TS -> local last nick TS (change) (for online time calculation)
nick->last
  last_signon -> from nick identify/register (local)
  first_TS -> from nick introduction (to be matched with user first_TS for validation)
